## Опис Рішення

Для гнучкості розширення Б.Д та данних для валідації використовувався підхід CodeFirst
 враховуючи що розмір Б.Д не великий. Також для гнучкості та простоти супроводження робота з запитами в Б.Д винесена
 в окремий класс.


## Перед запуском обов"язково виконати
У вас має бути встановлений МS SQL Server 2018 або будь-яка пізніша версія.
 Перед тим як запустити програму відкрийте Package Manager Console і напишіть add-migration "Init". Потім в
 Package Manager Console виконайте команду update-database і переконайтесь що з"явилась Б.Д з ім"ям DijaTestTaskDB.
 В сервісах є MockDataService з одним методом Fill для того щоб зекономити час і не заповнувати Б.Д вручну. Також в цьому
 класі ви можете подивитись які саме дані є в базі для тестування.

## Приклади тестування
Після запуску програми спочатку перконайтесь що в Б.Д є дані для тестування
В Консолі напишіть команду а потім натисніть ENTER. Доступні команди повністю з прикладом нижче:
passport СЮ112233
rnokpp 1234567890
birthday 01.01.2001
4cd20723-ddcf-4ace-8d4f-439f4431fead
